# 🌿 Paradise Nursery - e-plantShopping

An interactive React-based e-commerce application for buying houseplants. Users can browse plants by category, add items to a shopping cart, adjust quantities, and view total costs.

## Features
- Landing page with company info and “Get Started” button
- Product listing page with at least 6 plants in 3+ categories
- Add to Cart with button disabling after click
- Shopping cart with quantity increment/decrement, delete, and total calculation
- Redux state management for cart persistence
- Responsive header with dynamic cart count

## Technologies
- React 18
- Redux Toolkit
- React Router DOM
- Vite
- GitHub Pages deployment
